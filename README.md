# AI Projects

## 📌 Sentiment Analysis Project

This project performs sentiment analysis using Python.  
It analyzes text and classifies it into:

- 😊 Positive  
- 😐 Neutral  
- 😡 Negative  

---

## 🚀 Features
- Real-time text sentiment prediction  
- Simple and user-friendly  
- Uses NLP techniques  

---

## 🛠 Technologies Used
- Python  
- Streamlit  
- Transformers (Hugging Face)  

---

## ▶️ How to Run
1. Install required libraries:
   pip install transformers streamlit torch  

2. Run the project:
   streamlit run sentiment_analysis.py  

---

## 📊 Example
Input: "I love this product!"  
Output: Positive
